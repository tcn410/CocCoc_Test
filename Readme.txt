This repository is for CocCoc test.

Files in the repository:
    - Testing problem from CocCoc in pdf file.
    - Solution source code is in only 1 file, it is main.cpp.
    - This file (Readme.txt).


How to use the source code:
    - The source code is implemented to runs under Ubuntu 18.04 x64 OS, it has not been tested under other OSs yet.
    - Compile:
        On terminal, run the below command:
            g++ main.cpp -o execute_file_name
    - Run the executable file:
        On terminal, run a command similar to the below format:
            execute_file_name input_file output_file mem_limit_in_Bytes
